## Introduction
Covin은 최종적으로 코로나 검사를 받기까지 모든것을 자동화해줍니다.
1. 회원가입 과정에서 입력받은 정보를 통해 전자문진표를 자동으로 작성해줍니다.
2. 지도를 통해 코로나 검사가 가능한 장소를 파악할 수 있으며 사람이 얼마나 있는지를 확인할 수 있습니다/
3. 코로나 검사 예약 기능을 통해 예상 인원을 파악하고 인원을 분산시킬 수 있습니다.
